# 07 - Mini-Projects
Contains larger structured programs demonstrating basic system design and application of multiple concepts.

## 📝 Programs List

| # | File | Description |
|---|------|-------------|
| 1 | `BankingSystem.java` | Console banking app with accounts, deposits, withdrawals, and transfers |
| 2 | `QuizApplication.java` | Multiple-choice quiz engine with scoring and answer review |
| 3 | `StudentGradeManager.java` | Grade management with GPA, rank lists, and class statistics |
| 4 | `InventoryManagement.java` | Product inventory with stock tracking and low-stock alerts |
| 5 | `ContactBookApp.java` | Digital contact book with alphabetical ordering and regex validation |
| 6 | `PasswordGenerator.java` | Secure password generator with strength meter and batch mode |
| 7 | `NumberGuessingGame.java` | Guessing game with difficulty levels and hot/cold hints |
| 8 | `TicTacToeGame.java` | Two-player Tic-Tac-Toe with score tracking |
| 9 | `ExpenseTracker.java` | Personal expense tracker with category and monthly summaries |
| 10 | `LibraryCatalog.java` | Library catalog with book/member management and borrowing |

## 🧠 Concepts Covered
- Object-Oriented Design (encapsulation, composition, inner classes)
- Collections Framework (`HashMap`, `TreeMap`, `ArrayList`, `EnumMap`)
- Custom Exceptions and exception handling
- Input validation and regex patterns
- Streams and functional operations
- Menu-driven console application design
- Formatted tabular output
