# 01 - Basics
Contains fundamental Java logic, loops, and pattern matching programs.

## 📝 Programs List

1. **[PrimeFactorization.java](./PrimeFactorization.java)** - Computes the prime factors of a number efficiently.
2. **[PascalTriangle.java](./PascalTriangle.java)** - Prints Pascal's triangle up to N rows using combinations math instead of arrays.
3. **[ArmstrongNumberRange.java](./ArmstrongNumberRange.java)** - Finds all Armstrong numbers within a given range.
4. **[BaseConverter.java](./BaseConverter.java)** - Converts decimal numbers to any base (2-16) and vice versa.
5. **[DiamondPattern.java](./DiamondPattern.java)** - Generates a complex diamond pattern using nested loops.
6. **[CollatzConjecture.java](./CollatzConjecture.java)** - Prints the Collatz sequence and counts the steps.
7. **[TwinPrimes.java](./TwinPrimes.java)** - Finds and prints twin prime pairs up to a given limit.
8. **[PerfectNumberRange.java](./PerfectNumberRange.java)** - Finds all perfect numbers within a specified range.
9. **[HollowHourglassPattern.java](./HollowHourglassPattern.java)** - Generates an advanced hollow hourglass pattern.
10. **[EuclideanAlgorithm.java](./EuclideanAlgorithm.java)** - Finds GCD and LCM using the iterative Euclidean algorithm.
