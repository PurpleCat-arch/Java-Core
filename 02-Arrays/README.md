# 02 - Arrays
Contains all array-related programs, intermediate algorithms, and problem-solving techniques.

## 📝 Programs List

1. **[KadanesAlgorithm.java](./KadanesAlgorithm.java)** - Finds the contiguous sub-array with the maximum sum efficiently.
2. **[RotateArray.java](./RotateArray.java)** - Rotates an array to the right by k steps in O(n) time and O(1) space.
3. **[TrappingRainWater.java](./TrappingRainWater.java)** - Computes how much water can be trapped between elevation bars.
4. **[DutchNationalFlag.java](./DutchNationalFlag.java)** - Sorts an array of 0s, 1s, and 2s in-place (Dutch National Flag problem).
5. **[MissingAndRepeatingNumber.java](./MissingAndRepeatingNumber.java)** - Finds the missing and repeating numbers in an array using an in-place approach.
6. **[LongestConsecutiveSequence.java](./LongestConsecutiveSequence.java)** - Finds the length of the longest consecutive elements sequence in O(n) time.
7. **[TwoSum.java](./TwoSum.java)** - Finds two numbers that add up to a specific target number using a HashMap.
8. **[MajorityElement.java](./MajorityElement.java)** - Finds the majority element using Moore's Voting Algorithm.
9. **[SubarraySumEqualsK.java](./SubarraySumEqualsK.java)** - Finds the total number of continuous subarrays whose sum equals k.
10. **[MergeOverlappingIntervals.java](./MergeOverlappingIntervals.java)** - Merges all overlapping intervals into a consolidated set.
