# 04 - OOPs
Contains Object-Oriented Programming concepts: Classes, Objects, Constructors, Inheritance, Polymorphism, Encapsulation, Abstraction.

## 📝 Programs List

1. **[ClassAndObject.java](./ClassAndObject.java)** - Demonstrates the creation of a basic class and instantiating an object.
2. **[DefaultConstructor.java](./DefaultConstructor.java)** - Shows how a default constructor is invoked during object creation.
3. **[ParameterizedConstructor.java](./ParameterizedConstructor.java)** - Demonstrates a constructor that takes parameters to initialize object state.
4. **[CopyConstructor.java](./CopyConstructor.java)** - Illustrates creating a new object as a copy of an existing object.
5. **[ConstructorOverloading.java](./ConstructorOverloading.java)** - Shows how a class can have multiple constructors with different parameter lists.
6. **[MethodOverloading.java](./MethodOverloading.java)** - Demonstrates compile-time polymorphism by creating multiple methods with the same name but different parameters.
7. **[SingleInheritance.java](./SingleInheritance.java)** - Illustrates a child class inheriting from a single parent class.
8. **[MultilevelInheritance.java](./MultilevelInheritance.java)** - Shows an inheritance chain where a class inherits from a derived class.
9. **[HierarchicalInheritance.java](./HierarchicalInheritance.java)** - Demonstrates multiple child classes inheriting from a single parent class.
10. **[MethodOverriding.java](./MethodOverriding.java)** - Shows run-time polymorphism where a child class provides a specific implementation of a parent method.
11. **[SuperKeyword.java](./SuperKeyword.java)** - Explains how to use the `super` keyword to access parent class members and methods.
12. **[ThisKeyword.java](./ThisKeyword.java)** - Shows how to use the `this` keyword to refer to the current object instance.
13. **[StaticVariables.java](./StaticVariables.java)** - Demonstrates how static variables are shared among all instances of a class.
14. **[StaticMethods.java](./StaticMethods.java)** - Shows how static methods can be called without creating an instance of the class.
15. **[StaticBlock.java](./StaticBlock.java)** - Illustrates the use of a static block for static initialization.
16. **[FinalVariable.java](./FinalVariable.java)** - Demonstrates that a final variable's value cannot be changed once assigned.
17. **[FinalMethod.java](./FinalMethod.java)** - Shows that a final method cannot be overridden by subclasses.
18. **[FinalClass.java](./FinalClass.java)** - Illustrates that a final class cannot be inherited.
19. **[AbstractClass.java](./AbstractClass.java)** - Demonstrates the concept of abstraction using an abstract class and method.
20. **[InterfaceBasic.java](./InterfaceBasic.java)** - Shows how to define and implement a basic interface.
21. **[MultipleInheritanceUsingInterfaces.java](./MultipleInheritanceUsingInterfaces.java)** - Illustrates achieving multiple inheritance through interfaces in Java.
22. **[DefaultMethodInInterface.java](./DefaultMethodInInterface.java)** - Demonstrates default methods introduced in Java 8 for interfaces.
23. **[StaticMethodInInterface.java](./StaticMethodInInterface.java)** - Shows static methods within interfaces.
24. **[EncapsulationDemo.java](./EncapsulationDemo.java)** - Illustrates data hiding by keeping variables private and providing public accessors.
25. **[AccessModifiers.java](./AccessModifiers.java)** - Demonstrates public, protected, default, and private access modifiers.
26. **[GetterSetter.java](./GetterSetter.java)** - Shows standard practices for getting and setting private field values.
27. **[PolymorphismWithReferences.java](./PolymorphismWithReferences.java)** - Demonstrates dynamic method dispatch using parent references for child objects.
28. **[UpcastingDowncasting.java](./UpcastingDowncasting.java)** - Explains object type casting up and down the inheritance hierarchy.
29. **[AnonymousInnerClass.java](./AnonymousInnerClass.java)** - Shows how to instantiate a class without giving it a specific name.
30. **[NestedClasses.java](./NestedClasses.java)** - Demonstrates inner classes and static nested classes.
