Demonstrates a HashSet rejecting duplicate values.\n11. **[HashSetMethods.java](./HashSetMethods.java)** - Shows methods like remove, contains in HashSet.\n12. **[LinkedHashSetBasic.java](./LinkedHashSetBasic.java)** - Demonstrates a LinkedHashSet maintaining insertion order.\n13. **[TreeSetBasic.java](./TreeSetBasic.java)** - Shows how a TreeSet automatically sorts its elements.\n14. **[TreeSetMethods.java](./TreeSetMethods.java)** - Illustrates navigational methods in TreeSet.\n15. **[TreeSetCustomSorting.java](./TreeSetCustomSorting.java)** - Demonstrates a TreeSet using a Comparator for custom sorting.\n16. **[PriorityQueueBasic.java](./PriorityQueueBasic.java)** - Demonstrates a PriorityQueue processing elements based on priority.\n17. **[ArrayDequeBasic.java](./ArrayDequeBasic.java)** - Shows the usage of ArrayDeque as a double-ended queue.\n18. **[ArrayDequeAsStack.java](./ArrayDequeAsStack.java)** - Illustrates navigational methods in TreeMap.\n25. **[TreeMapCustomSorting.java](./TreeMapCustomSorting.java)** - Demonstrates a TreeMap using a Comparator to sort keys.\n26. **[HashTableBasic.java](./HashTableBasic.java)** -
