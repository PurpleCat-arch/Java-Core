# 03 - Strings
Contains all string manipulation and algorithm programs.

## 📝 Programs List

1. **[ValidAnagram.java](./ValidAnagram.java)** - Checks if two strings are valid anagrams using a character frequency array.
2. **[LongestSubstringWithoutRepeating.java](./LongestSubstringWithoutRepeating.java)** - Finds the length of the longest substring without repeating characters using the sliding window approach.
3. **[ValidPalindrome.java](./ValidPalindrome.java)** - Determines if a string is a palindrome considering only alphanumeric characters and ignoring cases.
4. **[LongestPalindromicSubstring.java](./LongestPalindromicSubstring.java)** - Finds the longest palindromic substring using the expand-around-center approach.
5. **[GroupAnagrams.java](./GroupAnagrams.java)** - Groups anagrams together from an array of strings using a HashMap.
6. **[StringCompression.java](./StringCompression.java)** - Compresses a string in-place by replacing consecutive repeating characters with their count.
7. **[FirstUniqueCharacter.java](./FirstUniqueCharacter.java)** - Finds the index of the first non-repeating character in a string using a frequency array.
8. **[ReverseWordsInString.java](./ReverseWordsInString.java)** - Reverses the words in a given string and handles multiple spaces correctly.
9. **[CheckStringRotation.java](./CheckStringRotation.java)** - Checks if one string is a rotation of another using string concatenation.
10. **[KMPAlgorithm.java](./KMPAlgorithm.java)** - Implements the Knuth-Morris-Pratt pattern searching algorithm to efficiently find occurrences of a pattern in a text.
